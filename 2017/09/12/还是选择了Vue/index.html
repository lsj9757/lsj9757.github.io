还是选择了Vue~

发表于 2017-09-12

感受到前端框架的强大。。。

-vue在用相同组件赋值的时候，一定要注意不能直接data:{}。。。要data:function(){ return{xxx：1} }不然可能会改变另一个同名组件的值
…

-vue.component()添加 全局 组件的方法

-对象比如:list:[ {name:’apple’,price:12},{name:’banana’,price:16} ],用v-for=”(item,index) in list” , ..

-数组比如:objlist:{ name:’apple’,price:35,color:’red’ },用v-for=”(value,key) in objlist” ,NaN…

-常用：push()-pop()-shift()-unshift()-splice()-sort()-reverse()..不触发（没用）: filter()-concat()-slice()

-不能强行改this.data…,可以用Vue.set(注意要引入Vue)

-用this.$emit调用组件里的函数（第三章5-7节）

-slot（插槽：第四章1-5节21分钟有讲）还行，keep-alive….可以缓存router页面

-过渡(如果是2个标签来回变，同样标签名（p与p）不行，没有动画，可以加个key=”xxx”来区别)有点烦。。

-vue里用let声明变量外层取不到{ let a=XXX } 取不到a

-可以用router.push等等编程式导航来自动跳转某个url

-用vuex里的commit方法调用全局vuex的函数(commit -> mutations)dispatch ->actions(第五章9-10节)

-图片需要通过JS来引入（放到script里时）就需要用到Require ！！！！

-vue里由父级给子组件传的prop值不能通过在子组件里修改再传给父级，而应该通过调用$emit父级事件在父级里面更改这个值/!!
